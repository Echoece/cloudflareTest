# Doctor Prescription (single page)

A tiny React + Vite app to test **Cloudflare Pages** deploys. It renders one page showing **patient history**, **drug history**, and **test history**.

## Node version

This project is pinned to Node **v24.13.0** via `.nvmrc`.

```bash
nvm install 24.13.0
nvm use 24.13.0
node -v
```

## Run locally

```bash
npm install
npm run dev
```

## Build (Cloudflare Pages)

- **Build command**: `npm run build`
- **Build output directory**: `dist`
